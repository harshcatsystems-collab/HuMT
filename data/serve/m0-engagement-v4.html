Hypothesis Testing (With Available Data)

❌ Hypothesis 1: Mix Shift — Direct Subscriber Base Shrinking

The Theory

Direct subscribers (Plan IDs with >90% watcher%) have 95%+ engagement. If this cohort is shrinking as % of M0 base while trial-converted cohort (with lower watcher%) is growing → overall M0 watcher% drops even if both cohorts perform the same.

V4 Status: LIKELY TRUE (Cannot Fully Test)

Circumstantial evidence:

  • T0 watcher% collapsed 81% (43% → 8%) from Nov → Feb
  • T0 = mix of direct subs (95% watcher%) + low-intent trials (10% watcher%)
  • If direct subs dropped from 50% → 10% of T0, watcher% would drop from 52% → 14% ✅ matches observed 8%

Need: Subscription Plan ID breakdown by week to confirm

❌ Hypothesis 2: Web Trial Growth — Low-Quality Cohort Expanding

The Theory

Web trials have 80.5% ghost rate (V3). If web trials grew from 15% → 35% of M0 base, that alone would increase blended ghost rate by ~8pp.

V4 Status: DISPROVEN

Evidence: Web watcher contribution stayed flat at ~1% across all months (Nov: 0.69%, Feb: 1.06%). If web trial volume had spiked, we'd see web watcher% spike proportionally. Didn't happen.

Conclusion: Web trials are poison, but they didn't grow. Not the cause of Feb spike.

❌ Hypothesis 3: Engagement Tier Shift — More T0/T1 (Low-Intent) Users

The Theory

If acquisition shifted toward users who don't engage during trial (T0/T1) and away from high-intent users (T4-T6), blended watcher% would drop.

V4 Status: INSUFFICIENT (But Unlikely)

Problem: We don't have engagement tier volume distribution (what % of M0 base is T0 vs T6?). We only have watcher% by tier.

But: Even if T0 grew from 60% → 80% of base, it wouldn't explain T0 watcher% dropping 81%. The tier itself degraded.

✅ Hypothesis 4: Temporal Quality Degradation — Acquisition Channels Got Worse

The Theory

Same acquisition channels (organic, FB ads, etc.) but declining quality within each channel. E.g., organic watcher% dropped from 40% → 25% over time, even though organic % of total volume stayed the same.

V4 Status: ✅ CONFIRMED (Primary Root Cause)

Evidence across ALL tiers:

  • T6 (best users): 74% → 65% watcher% (-12%)
  • T3 (mid users): 55% → 45% watcher% (-18%)
  • T0 (largest cohort): 43% → 8% watcher% (-81%)

Conclusion: Within-channel quality collapsed. The same T6 user profile (6 days active during trial) now converts at 12% lower rate than 3 months ago.

✅ PRIMARY ROOT CAUSE IDENTIFIED (CORRECTED March 3): Post-Subscription Engagement Drop

Note: Original V4 concluded "Acquisition Source Quality Collapse." Vismit + Kawaljeet correction + app open analysis proved this wrong. Trial engagement (T6 users) proves acquisition worked. Problem is post-subscription.

The Feb 2026 crisis is driven by two simultaneous failures:

  1. Direct subscriber base collapse (likely): T0 watcher% dropped 81% → suggests direct subs went from 50% of T0 → 10% of T0
  2. Post-subscription engagement drop (confirmed via app open analysis): Users who engaged during trial (T6: 3+ watch days) are opening app 2-3% less post-subscription. App opens declined across all tiers. This is NOT acquisition quality — it's subscription-period disengagement.

Combined impact: Fewer high-quality direct subs + trial-engaged users disengaging post-subscription = catastrophic blended watcher% collapse (7.21% → 4.89%, -32%).

Next diagnostic: Determine if app open drop is organic (content mismatch) or reengagement-driven (campaigns stopped). Requires push/email/retargeting data Dec vs Feb.

V3 said: "Web trials are the problem (80% ghost rate), paid ads work 2x better than organic."
V4 says: "Acquisition quality collapsed across EVERY tier (T0: -81%, T6: -12%). Root cause: Direct subscriber base shrinking + within-tier quality degradation. Fix sourcing, not conversion."

Same data. Different lenses. Completely different conclusions.

Five Principles I'll Carry Forward

  1. Start with the data model, not intuition. I assumed "web" meant web ads. It meant web platform. Data models are ground truth.
  2. Challenge the metric before optimizing it. Spent V1 analyzing watch conversion deeply without asking "is this the right metric?"
  3. Small segments can reveal big insights. Dismissed web trials as "only 1.3%" in V1. They were 25%. Never dismiss without validating measurement.
  4. Aggregate metrics hide more than they reveal. 65% ghost rate felt like one problem. Segmented by tier, it's 7 different problems with different causes.
  5. Temporal analysis catches degradation early. Without month-by-month tracking, we'd still think ghost rate is 60%. It's 76% and accelerating.

Part 10: App Open Analysis — The Course Correction

Post-V4 Analysis (March 3, 15:00 UTC)

Vismit Bansal and Kawaljeet Singh challenged V4's "acquisition quality collapse" conclusion. Their feedback led to this diagnostic analysis, which fundamentally changes our understanding of the crisis.

The Challenge from Vismit + Kawaljeet

Kawaljeet: "if the user is watching during the trial and not performing upto the mark in current subscription. Then there is no way we can take trial performance as a benchmark in defining user quality."

Vismit: "your analysis says this is acquisition quality problem which we both dont agree to."

The logical flaw they caught: If users engaged during trial (T6 = 3+ watch days), that PROVES acquisition brought engaged users. Post-subscription disengagement ≠ acquisition failure.

What This Proves

1. Vismit + Kawaljeet were right about acquisition:
T6+ tier (trial-engaged users) shows 38% app opens. These users watched 3+ days during trial — they're NOT "low quality." Acquisition brought engaged users.

2. The problem is post-subscription:
Even trial-engaged users (T6) are opening the app 2.2% less post-subscription. This is a subscription-period engagement drop, NOT an acquisition quality problem.

3. This is app open drop, not funnel:
All tiers declined. If it were a funnel problem (users open but don't watch), app opens would be stable. Instead, users aren't opening the app as frequently.

4. Next diagnostic question: WHY did app opens drop?

  • Hypothesis B: Reengagement campaigns reduced (push/email/retargeting)
Need reengagement data (Dec vs Feb) to isolate.
