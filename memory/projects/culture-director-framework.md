## What Is This?

The Culture Director is a **multi-agent AI system** that takes any story (screenplay, synopsis, treatment, produced film) in any culture and transforms it into an authentically adapted version in any target culture.

**It does NOT translate. It does NOT dub.**

It deconstructs, neutralizes, and reconstructs at every layer:
- Visible surface
- Invisible grammar
- Ideological substrate
- Material condition
- Kinship structure
- Language
- Memory
- Gaze

```
DECODE PHASE             NEUTRALIZE PHASE           ENCODE PHASE
```

## Governing Principle

**Agents are divided by ANALYTICAL MODE, not content domain.**

Every cultural element passes through multiple agents, each analyzing from a completely different cognitive lens. A single charpai gets analyzed by:
- Ideology Agent → authority structure it naturalizes
- Material Agent → economic condition that produced it
- Surface Agent → what it looks like, color, wear
- Deep Agent → habitus it signals (who reclines, who perches)
- Language Agent → dialogue around it
- Kinship Agent → who sits where by obligation
- Memory Agent → narrative it carries across generations
- Gaze Agent → camera romanticizing (tourist) or assuming (insider)?

## The 8 Forms of Culture

| Form | Description |
|------|-------------|
| **DOMINANT** | Hegemonic, mainstream — "normal" without saying it |
| **FOLK** | Community-based, oral, participatory — ragini, birha, Chhath geet |
| **TRADITIONAL** | Formally preserved — khap panchayat, Rajput ceremony |
| **POP** | Contemporary popular — web series, Haryanvi rap, Instagram reels |
| **WORKING CLASS** | Labor culture — farm workers, migrants, factory floor |
| **MASS** | Homogenized commercial — Bollywood, OTT, brand aspirations |
| **REGIONAL** | Geographically specific — Bhojpuri ≠ Awadhi ≠ Maithili |
| **COUNTER** | Resistant, alternative — inter-caste love, queer presence |

**Key insight:** Every real community contains ALL 8 forms simultaneously. A story drawing from only one form is a stereotype, not a culture.

### Two-Level Classification

1. **MACRO (Phase 0):** Story-wide Forms Profile at intake — orients all agents
2. **MICRO (Phase 1):** Element-level tagging by each agent — reveals forms tension within scenes

## Agent Architecture (12 Total)

### Layer 1: Orchestrator
| Agent | Role |
|-------|------|
| **Agent 0: Culture Director** | Orchestrator, synthesizer, quality owner, conflict resolver |

### Layer 2: Analytical (8 Bidirectional Agents)
| Agent | Theoretical Lineage | Core Question (Decode) |
|-------|---------------------|------------------------|
| **1. Ideology Analyst** | Barthes → Gramsci → Williams → Althusser | What does this culture BELIEVE? |
| **2. Material Conditions** | Marx → Williams → Bourdieu | What CONDITIONS produced this? |
| **3. Surface Culture** | Geertz → Barthes | What is PERCEIVABLE here? |
| **4. Deep Culture** | Bourdieu → Merleau-Ponty → Sobchack | What can be FELT but not seen? |
| **5. Language & Dialogue** | Bakhtin → de Certeau → Austin | How does it SPEAK and leave UNSAID? |
| **6. Kinship & Social** | Lévi-Strauss → Dumont → Parry | WHO is obligated to WHOM? |
| **7. Memory & Narrative** | Williams → Benjamin → Nora | How does it REMEMBER itself? (8 Forms owner) |
| **8. Gaze & Spectatorship** | Mulvey → Hall → Madianou → Deger | WHO is watching, from WHERE? |

### Layer 3: Execution
| Agent | Role |
|-------|------|
| **Agent 9: Story Neutralizer** | Extracts universal story from cultural container |
| **Agent 10: Reconstruction Engine** | Writes adapted script (3-pass process) |

### Layer 4: Validation
| Agent | Role |
|-------|------|
| **Agent 11: Authenticity Validator** | 12-point checklist, Forms Balance, adversarial critique |

## Authenticity Validator — 12-Point Checklist

### Layer A Checks (6 points)
1. Character names and kinship terms authentic to pod
2. Key locations pod-specific (not generic)
3. Pod-specific food in 2+ scenes with cultural meaning
4. Sound landscape deployed in 2+ scenes
5. Visual objects from pod catalog in 3+ scenes
6. Ceremony follows pod's ritual template beats

### Layer B Checks (4 points)
7. Habitus authentic — bodies carry target culture's history
8. Affective registers correct — grief/joy/anger performed in pod-specific way
9. Dialogue register authentic — right rhythm, silence, address terms
10. IZZAT operates with pod-specific trigger, response, resolution

### Cross-System Checks (2 points)
11. No anti-pattern violations (tourist-trap, stereotypes)
12. All 8 Layer A categories represented across script

**Automatic fail:** Layer A present but Layer B absent = "Costume Party"

## Key Concepts

### Costume Party
When visible cultural markers (Layer A) are present but invisible grammar (Layer B) is absent. Looks like the culture but doesn't feel like it.

### The Remake Problem
Five things that never "dub":
1. Relationship tempo
2. Silence weight
3. Honor economy
4. Assumed audience knowledge
5. Affective register

### Forms Translation Decision (Phase 5)
**MAINTAIN:** Same forms profile, different cultural expression
**SHIFT:** Different forms profile that carries same/greater emotional weight

Agent 7 recommends; Director decides.

### Lifecycle Convergence Protocol
For lifecycle events (birth, marriage, death, etc.), ALL 8 agents fire simultaneously regardless of complexity rating.

## Problems This Architecture Solves

| Problem | Solution |
|---------|----------|
| Costume party adaptations | Agent 4 (Deep) + Validator detection |
| Tourist-gaze framing | Agent 8 (Gaze) monitoring |
| Culture as monolith | Agent 7 (Memory) + 8 Forms balance |
| Language inauthenticity | Agent 5 dedicated dialogue pass |
| Kinship structural errors | Agent 6 maps all obligations |
| Safe adaptations (only Dominant + Traditional) | Validator checks for Counter/Working Class presence |
| Tier 3 substitution failure | Story Neutralizer creates functional equivalence briefs |
