## North Star Metric

| Metric | Current | Target |
|--------|---------|--------|
| **Week 1 Watcher %** | 30% | 36% |

**Key Insight:** 70% of M0 watchers consume their first content in the first 7 days of subscription. Momentum drops off after that — hence Week 1 is the critical intervention window.

---

## Strategic Initiatives

### Initiative 1: Sharp Reach-Outs (Efficient & Focused)

Target: App open by subscription-active and app-installed users

| Pillar | Description |
|--------|-------------|
| **Behavioral Segmentation** | Target users based on behaviors and consumption patterns for precise outreach |
| **Content Match Making & Tonality** | Match content and tonality of comms to each user's consumption preferences |
| **Appropriate Moment of Targeting** | Reach users at the right time using activity signals and optimal engagement windows |
| **Targeting Channel** | Select most effective communication channel per user based on past behaviour |

**Execution focus:**
- Taste of content based on acquisition content and watch history
- Intent-based targeting
- Churn probability scoring

---

### Initiative 2: Tambola

**Tagline:** "Play, Relax, Win While Watching"

| Focus Area | Plan | Type |
|------------|------|------|
| **Daily Tambola Rollout** | Launch daily Tambola games to build habit loop — keep Sunday as flagship event with bigger rewards | Habit Loop |
| **On-App Visibility** | Create persistent Tambola entry point on Home + contextual nudges during content consumption | Always-On CTA |
| **Marketing Support** | Drive discovery via pre-roll ads, Meta ads, Google ads, push notifications, in-app pop-ups, homepage placements — position Tambola as daily entertainment moment | Multi-Channel Push |
| **Dashboard & Tracking** | Build holistic Tambola dashboard tracking opens, registrations, participation, repeat users for continuous optimisation | Data-Driven |

**Execution variants:**
1. Tambola / Bingo everyday
2. 15 numbers everyday (daily variant)
3. Improve production quality — Celebrity approach
4. Marketing approach — perf ads, pre-roll, app visibility, communication

---

### Initiative 3: Character Bots as Re-engagement Vehicle

**🔗 Links to Pranay's PIP strategic bet (Character Bots)**

| Pillar | Implementation |
|--------|----------------|
| **Behavioral Segmentation** | Bot Selection by Watch Behavior — assign right character bot to each M0 user based on watch history |
| **Content Match Making & Tonality** | Bot Speaks in Character Voice — each bot uses tone, humour, vocabulary of the character it represents. Messages match content type |
| **Appropriate Moment of Targeting** | Bot Triggers at Right Window — bots initiate chat when M0 users are most receptive, timed to user activity |
| **Targeting Channel** | In-App Chat as Primary Channel — character bot itself becomes a channel |

**Key insight:** Bots become a re-engagement channel, not just a feature.

**Execution:**
- Distribution of Chatbot to M0 cohort

---

### Initiative 4: Web Consumption Focus

**Target:** Reactivate uninstalled M0 subscribers through frictionless web experiences

| Feature | Benefit |
|---------|---------|
| **One Tap Login** | Eliminate friction in auth flow. Enable sign-in with single tap, reducing drop-off at entry point |
| **Watch Without Login** | Allow uninstalled M0 subscribers to consume content on web without login. Lower barrier to re-engagement |
| **PWA (Progressive Web App)** | Deliver native-like app experience via web browser. Enable install prompts, push notifications, offline content |

**Execution:**
- One Tap Login
- PWA implementation
- Free Access Distribution
