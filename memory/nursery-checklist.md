# 🏠 Nursery Setup Checklist

**Target Completion:** Week 32 (June 18, 2026)

---

## Philosophy

The nursery doesn't need to be Pinterest-perfect. It needs to be:
- **Safe** — Baby can't hurt themselves
- **Functional** — You can change, feed, soothe at 3 AM half-asleep
- **Calming** — Soft colors, minimal clutter, restful vibe

You can always upgrade later. Start practical.

---

## 🛏️ Sleep Zone (Priority #1)

### Crib / Bassinet
- [ ] Crib (convertible cribs grow with baby) OR
- [ ] Bassinet (for first 3-4 months, can keep in your room)
- [ ] Firm mattress (fits snugly, no gaps)
- [ ] 2-3 fitted crib sheets (cotton, breathable)
- [ ] Waterproof mattress protector

**Safety Rules:**
- Nothing in the crib except baby + fitted sheet
- No pillows, blankets, bumpers, stuffed animals
- Baby sleeps on back, always
- Crib away from windows, cords, curtains

### Nice to Have
- [ ] White noise machine (helps sleep SO much)
- [ ] Dim nightlight (for night feeds without waking fully)
- [ ] Baby monitor (video or audio)
- [ ] Blackout curtains (daytime naps)

---

## 🧷 Diaper Changing Zone (Priority #2)

### Changing Station
- [ ] Changing table OR dresser with changing pad on top
- [ ] Contoured changing pad with safety strap
- [ ] 2-3 changing pad covers (waterproof or washable)

### Supplies (Arm's Reach)
- [ ] Diapers (newborn + size 1)
- [ ] Wipes (fragrance-free, sensitive skin)
- [ ] Diaper rash cream (Sudocrem, Desitin)
- [ ] Diaper disposal system (Diaper Genie) OR
- [ ] Lined trash can with lid
- [ ] Hand sanitizer
- [ ] Extra set of clothes (onesie, diaper) kept nearby

**Pro Tip:** Set up a SECOND changing station in living area. You don't want to run to nursery every time.

---

## 🪑 Feeding & Comfort Zone

### Seating
- [ ] Comfortable chair / glider / recliner
- [ ] Supportive pillow (for nursing or bottle feeding)
- [ ] Small side table (for water, phone, burp cloth)
- [ ] Footrest (optional but helps posture)

### Feeding Supplies
- [ ] Nursing pillow (Boppy, My Brest Friend)
- [ ] Burp cloths (at least 6-8)
- [ ] Bottles (if supplementing or pumping)
- [ ] Bottle warmer (if using bottles)
- [ ] Breast pump (if planning to pump)

## 👕 Clothing & Storage

### Dresser / Wardrobe
- [ ] Dresser with drawers OR closet organizers
- [ ] Drawer dividers (sort by size: NB, 0-3, 3-6 months)
- [ ] Hangers (small, baby-sized)
- [ ] Hamper (for dirty clothes)

### Clothing Essentials (0-3 months)
- [ ] Onesies / bodysuits (8-10)
- [ ] Sleepers / footed pajamas (5-6)
- [ ] Pants / leggings (4-5)
- [ ] Socks / booties (6-8 pairs)
- [ ] Mittens (scratch mitts) (3-4 pairs)
- [ ] Caps / beanies (3-4)
- [ ] Swaddle blankets (4-6)
- [ ] Light jacket / sweater (2)

**Note:** Don't overbuy newborn size. They outgrow it in weeks. Stock more 0-3 and 3-6 month sizes.

## 📦 Shopping Strategy

### Phase 1: Research (Week 20-24)
- Browse online, read reviews
- Visit baby stores, see in person
- Ask parent friends what they actually used

### Phase 2: Order (Week 24-28)
- Order major items (crib, dresser, chair)
- Allow time for delivery delays
- Don't open everything yet (in case of returns)

### Phase 3: Setup (Week 28-32)
- Assemble furniture
- Wash all clothes, sheets, towels
- Organize and arrange
- Do a "test" — can you reach everything in the dark?

### Phase 4: Final Touches (Week 32-36)
- Add supplies (diapers, wipes, creams)
- Stock up on essentials
- Install monitor, test it

## 🔒 Safety Checklist (Final Review)

Before baby arrives, walk through:

- [ ] Crib meets safety standards (slat spacing, no drop sides)
- [ ] Nothing soft in crib
- [ ] Furniture anchored to wall (anti-tip)
- [ ] No cords near crib (blinds, chargers, monitors)
- [ ] Smoke detector in/near room
- [ ] Room temperature comfortable (20-22°C ideal)
- [ ] No direct AC/fan on crib
- [ ] Night feeding path clear (no tripping hazards)
