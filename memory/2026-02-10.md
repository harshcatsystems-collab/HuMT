### Problem
HMT messaged on WhatsApp during lunch, never got a response.

### Root Cause
**Status 408 = Request Timeout** — WhatsApp Web WebSocket connections drop when:
- Mac goes idle/sleeps
- Network hiccups occur

### Disconnect Timeline (during HMT's lunch)
- 14:43 — disconnect, reconnect in 6s
- 15:01 — disconnect, reconnect in 4s
- 15:18 — disconnect, reconnect in 4s

Messages sent during disconnect windows get lost.

### Slack Also Affected
Pong timeouts observed — same root cause (WebSocket keepalive failures on idle)

### Potential Fixes
1. **caffeinate -d &** — prevent Mac sleep (quick fix)
2. Run on always-on server (VPS/Pi) — permanent fix
3. Check WiFi stability
4. Cron job to alert if WhatsApp drops >5 min
