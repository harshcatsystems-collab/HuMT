## Scoring Rules
- **👍 / positive reaction / acts on it** → +1 (useful)
- **Ignored / no reaction** → 0 (neutral)
- **"I knew that" / "obvious"** → -1 (too noisy)
- **"Why didn't you tell me?" / surprise** → -2 (MISSED — worst case)
- **"Too long" / skimmed** → flag for compression
