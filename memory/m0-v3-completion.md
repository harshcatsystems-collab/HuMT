### 2. Key Learnings Documented
**Learning #1 (from Nikhil):** Wrong segment definition
- Used acquisition source instead of platform field
- Web trials = 25% not 1.3%
- Lesson: Validate segment definitions against data model first

**Learning #2 (from Vismit):** Wrong metrics entirely
- Was measuring funnel conversion (watch conversion 3.09%)
- Should measure cohort quality (Watcher% + Habit%)
- Lesson: Challenge the metric before optimizing it

**Where feedback overlaps:** Both said "stop aggregating, start segmenting"
- Nikhil: Segment by platform
- Vismit: Segment by channel, time, intent
- Unified: STAGE has 15 different problems, not one

### 3. Key Findings (v3)
- **Ghost rate worsening:** 60% (Sep) → 76% (Feb) — crisis, not steady-state
- **Channel quality spread:** Facebook Ads 54.8% Watcher% vs Organic 33.5%
- **Paid acquisition works:** 1.5x higher Watcher%, 2.3x higher Habit%
- **Web trials toxic:** 80.5% ghost rate, destroying blended metrics
- **App-installed cohort healthy:** 57.7% Watcher% (2x better than blended)
