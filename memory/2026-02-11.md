### Lesson Learned
- When OpenClaw is freshly configured (`openclaw configure`), channel dmPolicy defaults to "pairing" — need to explicitly set allowlist + allowFrom for each channel.
- Slack DM policy lives at `channels.slack.dm.policy` (nested), not `channels.slack.dmPolicy` like Telegram/WhatsApp.
- Cross-channel messaging from Telegram→Slack is blocked by security; use sub-agents to send to other channels.
