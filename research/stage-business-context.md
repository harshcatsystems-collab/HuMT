## 4. Content Strategy

### Content Philosophy
- Shifted from "culture-first entertainment" to "entertainment-first in cultures"
- Content formats: Feature Films, MicroDramas, Web Series, Short Films, Documentaries, Mini content
- MicroDramas: Pioneered hyperlocal micro-drama on OTT — library grew from 53 to 97+ titles, driving 17% of title starts
- Weekly episodic release format adopted (e.g., Maha-Punarjanam 26 episodes)
- Content release frequency: ~6 titles/month across all markets

## 5. Product & Technology

### Key Product Areas
- **Acquisition/Activation:** Web contributing ~20% of trials; 7-day trial → ₹199/month plan; festive discount offers
- **Engagement:** MicroDramas driving higher session frequency (1.8 vs 1.3 daily sessions)
- **Retention:** M0 watcher % grew 46% → 50%; Habit % from 4.16% → 6.27%; WoW retention 45% → 52%
- **TV Platform:** Android TV, Samsung TV, LG TV apps; Jio STB pre-install on 12M existing STBs + 20M new smart TVs; Weekly active TV users grew to 5.8K
- **Personalization:** Currently NO recommendation system — single biggest gap. All users see same homepage.
- **Payments:** State-of-art queue-based processing; UPI autopay stack (PhonePe, Paytm, PayU, Yesbizz, Razorpay, Setu)

### Key Metrics Tracked
- AHA moment (4 active days in first 7) — currently 8-10%, target 25%
- M0 watcher %, M1 retention (currently ~22%, target 35%)
- WAU%, MAU%, WoW retention
- Trial conversion (signup → trial → subscription)
- Watch time rate (WTR), Completion rate (CR), Hook rate
- CAC, LTV, ROAS
- Daily resurrected users, Habit %

### Technology Stack
- A/B testing infrastructure (Statisg platform)
- Infrastructure optimized for reduced bandwidth (~25% reduction)
- Google TV Content Recommendations integration
- ONDC integration (buyer + seller app)

## 6. AI Transformation Strategy (Feb 2026 — THE Current Priority)

This is the defining initiative right now. 14 interconnected AI projects forming an "AI-native operating system":

### Three Layers
**Foundation (Layer 1):**
- Project 14: Data & Instrumentation Cleanup
- Project 05: Reena (Common Brain / Knowledge Graph)
- Project 01: BMAD Framework (AI agents for every function)

**Intelligence (Layer 2):**
- Project 07: AI Native Consumer Insight
- Project 09: Performance Marketing Framework
- Project 10: 1 User = 1 Agent (Acquisition)
- Project 11: STAGE Saathi (Engagement Agent via WhatsApp)
- Project 12: 1 User = 1 Home Page (Personalization)

**Value Creation (Layer 3):**
- Project 02: Productising the Cultures (Cultural Adaptation)
- Project 03: Agentic IP Generation Framework
- Project 04: Social Marketing Automation
- Project 06: Animation Strategy for TAM
- Project 08: Title Marketing AI
- Project 13: Immersive Content Experience
- Project 15: AI-Native Character Bots

### 90-Day Roadmap
- **Phase 1 (Days 1-30):** Foundation — data quality, Reena MVP, BMAD pilot, 17 engineers active
- **Phase 2 (Days 31-60):** Intelligence — personalized homepage, Sarthi, LTV bidding, 23 engineers
- **Phase 3 (Days 61-90):** Scale — 15+ cultures, 3x content velocity, full automation

### Key Targets

- Same 25-person eng team, 10x output
- AHA moment: 8-10% → 25%
